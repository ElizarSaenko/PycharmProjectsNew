РАСПИСАНИЕ

Данный проект представляет собой веб-страницу с расписанием занятий и ежедневной активности. На странице представлены две таблицы: первая содержит расписание учебных предметов, а вторая - расписание для завтрака, спортзала, душа, ужина и экстра событий.

Реализованно:
1. **Таблица расписания занятий**:
   - Содержит дни недели, названия предметов и время их проведения.
   - Строки могут быть добавлены для дополнительных дней и предметов в соответствии с нуждами пользователя через файл "Home.htm". Функция "rowspan" позволит выставить нужное количество строк для предметов на текущий день


Ниже располагается следующаяя таблица.
2. **Таблица расписаия активности**:
   - Включает информацию о времени базовых дел для каждого дня недели. Делм можно поменять заголовки под нужды пользователя, для этого нужно дойти до обозначения второй таблицы (133 строки кода в текущей конфигурации программы) и заменить активности, заключенные с обоих сторон "<th> - </th>" на нужные вам.

 
Не реализована функция парсинга расписания с сайта и подключение базы дынных к веб-сайту.



**Описание работы**->->->


Начинаем с объявления типа документа:

<!DOCTYPE html>
Это определяет, что мой документ является HTML5.

Затем я открываю корневой элемент HTML и указываю язык документа:

<html lang="en">
В данном случае это английский.

- `<meta charset="UTF-8">` устанавливает кодировку документа на UTF-8, что позволяет использовать все символы, включая кириллицу.
- `<title>` устанавливает название вкладки браузера, которое будет отображаться.

Далее идет основной контент страницы, который виден пользователю, отмеченный тегом `<body>`:

<body>
    <h1>Расписание занятий</h1>
    <table>
        ...
    </table>
</body>

### CSS Раздел

Внутри тега `<style>` добавлем стили для элементов на странице, чтобы создать приятный визуальный эффект:

body {
    font-family: Arial, sans-serif;
    background-color: #f4f4f4;
    margin: 20px;
}
- Эти стили для `body` определяют шрифт, цвет фона и отступы.

h1 {
    text-align: center;
    color: #333;
}
- Здесь я задаю стиль для заголовка h1 (главного заголовка расписания): выравнивание по центру и цвет текста.

table {
    width: 100%;
    border-collapse: collapse; 
    margin-top: 20px;
}
- Устанавливаем размеры и оформление для таблицы, используя `border-collapse: collapse`

th, td {
    border: 1px solid #ccc;
    padding: 10px;
    text-align: left;
    transition: background-color 0.3s; / Плавный переход цвета /
}
- Эти стили применяются к заголовкам таблицы (th) и ячейкам (td): они определяют рамки, отступы, выравнивание текста и плавный переход цвета при изменении фона.

th {
    background-color: #007BFF;
    color: white;
}
- Я устанавливаю цвет фона и текста для заголовков таблицы.

.daych, .daynch {
    font-weight: bold;
    text-align: center;
    border-top: 2px solid #007BFF; / Верхняя граница для выделения /
}
- Эта часть кода отвечает за классы `daych` и `daynch`, задавая жирный шрифт, центровку и верхнюю границу для выделения дней недели.

td:hover {
    background-color: #d1e7dd; / Цвет при наведении /
}
- Меняет цвет фона ячейки, когда курсор наводится на неё.

### Таблица

Используем элемент `<table>` для создания таблицы расписания:

<table>
    <tr>
        <th>День</th>
        <th>Предмет</th>
        <th>Время</th>
    </tr>
    <tr>
        <td class="daynch" rowspan="2">Понедельник</td>
        <td>Базы данных</td>
        <td>11:50 - 13:20</td>
    </tr>
    <tr>
        <td>Базы данных</td>
        <td>13:30 - 15:00</td>
    </tr>
    ...
</table>

- Каждая строка таблицы начинается с `<tr>` (table row).
- Ячейки определяются с помощью `<td>` (table data), а заголовки — с помощью `<th>` (table header).
- Атрибут `rowspan="2"` указывает, что данная ячейка занимает две строки (например, для понедельника или вторника).
- Каждая ячейка заполняется информацией о дне, предмете и времени, её МОЖНО менять вручную под свое расписание и нужды.
